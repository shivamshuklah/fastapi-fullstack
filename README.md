
# ⚡ Shivam's Full Stack Web App (FastAPI + React)

A production-ready full-stack application using FastAPI for backend and React (with Chakra UI) for frontend. Secure, scalable, and Dockerized for deployment.

## 🔧 Features

- FastAPI backend with SQLModel & PostgreSQL
- React frontend with Chakra UI
- JWT-based authentication
- Email-based password recovery
- Dockerized deployment using Docker Compose
- GitHub Actions for CI/CD

## 🛠️ Tech Stack

- FastAPI, Python, SQLModel
- PostgreSQL
- React + TypeScript + Chakra UI
- Docker, GitHub Actions

## 🧪 Local Setup

```bash
git clone https://github.com/shivamshuklah/fastapi-fullstack-app.git
cd fastapi-fullstack-app
cp .env.example .env
docker-compose up --build
```
